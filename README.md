
# 🎓 Student Management System

## 📌 Overview

A simple and efficient **Student Management System** designed to manage student records with ease. This project demonstrates basic CRUD (Create, Read, Update, Delete) operations and is ideal for beginners learning software development.

---

## 🚀 Features

* ➕ Add new student details
* 📋 View all student records
* ✏️ Update student information
* ❌ Delete student data
* 🔍 Search students (optional if you have it)

---

## 🛠️ Tech Stack

* Language: Python
* Concepts Used: OOP, File Handling
---

## 📂 Project Structure

```
student-management/

## ▶️ How to Run

1. Clone the repository

   ```
   git clone https://github.com/priyakorada/student-management.git
   ```
2. Navigate to the project folder

   ```
   cd student-management
   ```
3. Run the project

   ```
   python main.py
   ```
